# Crop Prediction using Machine Learning
## About the Project
This project focuses on crop prediction in a method that determines which crop is best to grow using several significant parameters like:
- Nitrogen Value
- Phosphrous Value
- Potassium Value
- Temperature Range
- Humidity Content
- pH value
- Average Rainfall
